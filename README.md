# Python 基础准备 - 完成任务代码

## 使用方式

用 PyCharm 打开本文件夹，直接运行对应 `.py` 文件即可。

## 文件说明

- `00_test_env.py`：Python 环境测试
- `01_guess_game.py`：猜数字游戏
- `02_supermarket.py`：超市购物系统
- `03_family_accounting.py`：家庭记账系统
- `04_icbc_bank.py`：工商银行账户管理系统
- `05_methods_practice.py`：方法练习题
- `06_oop_practice.py`：面向对象练习题
- `07_data_report.py`：员工数据统计报表
- `08_database_sqlite_demo.py`：数据库任务可运行版
- `run_all_tests.py`：快速检查主要代码是否能运行
- `data/employees.csv`：员工统计数据

## 说明

代码尽量使用 Python 自带功能，减少第三方依赖。
其中数据库任务为了能直接在 PyCharm 运行，使用 Python 自带的 `sqlite3`，不需要安装 MySQL。
